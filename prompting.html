Prompting meistern

Ein guter Prompt ist keine Magie, sondern eine strukturierte Anweisung. Beantworten Sie die folgenden fünf Fragen, um die KI präzise zu steuern und qualitativ hochwertige Ergebnisse zu erhalten.

Die 5 Leitfragen für einen perfekten Prompt

1. Was soll entstehen? (Format)

Definieren Sie das Endprodukt. Z.B. ein Lektionsplan, eine E-Mail, eine Tabelle, ein Bild, ein Code-Snippet.

2. Für wen ist es bestimmt? (Rolle/Zielgruppe)

Bestimmen Sie das Publikum. Z.B. Lernende im Betrieb, Schulleitende, IKT-Verantwortliche.

3. Wofür wird es gebraucht? (Zweck)

Beschreiben Sie den Verwendungszweck. Z.B. als Einstieg, zur Prüfungsvorbereitung, für eine Präsentation.

4. Worauf soll sich die KI beziehen? (Input-Quelle)

Grenzen Sie den Wissensraum ein, indem Sie eine Quelle vorgeben. Z.B. eine Webseite, ein PDF, ein Text.

5. Wie soll es aussehen? (Output-Spezifikation)

Machen Sie Vorgaben zu Umfang, Sprachstil und medialer Ausgestaltung. Z.B. "kurz & prägnant", "formell", "multimedial".
